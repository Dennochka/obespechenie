## Ответы на вопросы
1. Параметр - техническая характеристика окружения, на котором проводится тестирование ПО. Они содержат значения и помогают избежать дублирования тест-кейсов. Для создания параметров: Войдиет в систему, с ролью **Администратор**, затем в верхней навигационной панели выберите **Настройки Проектов** -> В навигационном меню слева выберите  **Параметры**. Отобразится список параметров -> Нажмите **Создать параметр** -> Введите данные параметра (Название, Проекты, в которых параметр будет использоваться, Значения параметра) -> Нажмите **Сохранить** -> Чтобы отредактировать параметр, нажмите значок **Действия** справа от его названия, выберите **Редактировать**, затем внесите необходимые изменения и сохраните их.

2. Поле продолжительность отвечает за время прохождения тест-кейса.

3. Ссылки в проекте указываются для того, чтобы ссылаться на задачи, дефекты и связанные рабочие элементы, а также отслеживать статус связанных задач и дефектов. Для добавления ссылки: Откройте проект -> Откройте рабочий элемент, в который хотите добавить ссылку -> В навигационном меню слева перейдите в раздел **Ссылки** -> Введите тип и URL ссылки, которую хотите добавить в рабочий элемент -> Нажмите **Добавить**. ССылка отобразится в таблице.
